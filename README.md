# Sales Control
## Sistema de Ventas e Inventario
Proyecto de práctica API REST desarrollado con Spring Boot MVC

### Dependencias:
* JPA
* H2 DB 
* Spring Security
* Thymeleaf
 
### Características: 
* Autenticación
* Localización (ESP, EN, DE)
* Generación de reportes (json, xml, csv)
